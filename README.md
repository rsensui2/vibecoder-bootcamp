# VibeCoder育成プログラム

## 概要

VibeCoder育成プログラムは、非エンジニアがAI駆動開発のスキルを習得するための総合的な研修プログラムです。このリポジトリには、研修資料、講義内容、営業資料、および関連するドキュメントが含まれています。

## プログラムの特徴

- **AI駆動要件定義**: Claude、ChatGPTなどのLLMを活用した要件定義手法
- **実践的開発**: Cursor、v0、Bolt.newなどの最新AIツールを使用
- **段階的学習**: 基礎から実装まで体系的なカリキュラム
- **ハンズオン形式**: 実際のWebアプリケーション開発を通じた学習

## 技術スタック

### 開発ツール
- **Cursor**: AI統合開発環境
- **v0**: UIプロトタイピング
- **Bolt.new**: フルスタック開発
- **Claude/ChatGPT**: 要件定義・設計支援

### 実装技術
- **Frontend**: Next.js, React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, Prisma
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Clerk
- **Deployment**: Vercel

## カリキュラム

### 第1回: 基礎とマインドセット
- AI駆動開発の概念理解
- 各種AIツールの紹介
- 要件定義の基礎

### 第2回: 技術スタックと開発環境
- 開発環境の構築
- 技術スタックの理解
- AI駆動要件定義の実践

### 第3回: データベースと認証
- データベース設計
- Supabaseの活用

### 第4回: デプロイと運用
- Vercelでのデプロイ

## 学習目標

受講者は以下のスキルを習得できます：

1. **AI活用スキル**: 各種AIツールを効果的に活用した開発手法
2. **要件定義能力**: ビジネス要求をAIと協働で技術仕様に落とし込む能力
3. **実装スキル**: モダンなWebアプリケーションの開発・デプロイスキル
4. **プロジェクト管理**: Git/GitHubを使ったバージョン管理

## 対象者

- プログラミング経験が少ない非エンジニア
- AIツールを活用した開発に興味のある方
- 新しい技術を学習したいビジネスパーソン
- デジタル変革を推進したい企業の担当者

## 受講要件

- 基本的なコンピューター操作スキル
- 学習意欲と積極的な参加姿勢
- 各種AIツールのアカウント作成（研修内で案内）
