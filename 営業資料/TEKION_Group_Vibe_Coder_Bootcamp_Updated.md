## 1. なぜ今、ビジネスサイドにAI開発スキルが必要なのか？

### AI時代の学習アプローチの転換

- **従来の「個人の知識蓄積」型学習**
  - 個人の努力で知識を蓄積
  - 学習成果は個人の能力に依存
  - 成長に限界がある
  - 新技術ごとに学習し直しが必要

- **新しい「AI活用スキル」型学習**
  - AIの進化を自分の力にするマインドセット
  - 同じ努力でも成果が継続的に向上
  - 世界中のAI進化の恩恵をすべて享受
  - AIという"同僚"と共に成長

**生成AIの進化速度は止まらない。個人の学習意欲をAIの徹底活用前提に見直す時代**

### IDE：アイデアを現実に変える究極のプラットフォーム

Vibe Coder Bootcampが最も重視するのが、**IDE（統合開発環境）を使いこなすスキル**です。なぜなら、IDEこそが、AIの能力を最大限に引き出し、単なる文章作成ツールから「価値創造ツール」へと昇華させる究極のプラットフォームだからです。

- **チャットUIとの決定的な違い**
  - 従来のChatGPTのようなチャットサービスでは、扱える情報量（コンテキスト）に限界がありました。しかし、IDE上でAIを操作することで、プロジェクト全体の膨大なファイル群をAIに参照させ、より高精度で文脈に沿ったアウトプットを得られます。

- **世界の最先端を、あなたの追い風に**
  - 現在、GAFAを筆頭に世界のテックジャイアントがIDE市場で熾烈な競争を繰り広げています。この世界的な開発競争の恩恵を、非エンジニアであるあなたが直接享受し、自身の成長の追い風にできるのです。

- **職種を問わず応用できる「構築力」**
  - 「目的を定め、要件を定義し、設計し、形にする」というプロセスは、企画、マーケティング、営業など、あらゆるビジネス職に共通する思考法です。本ブートキャンプで学ぶエンジニアリングのプロセスは、最も複雑で応用範囲の広いスキルセットであり、一度習得すれば、あなたのビジネススキルは飛躍的に向上します。
  - 実際に、開発とは無縁だった受講生から「エンジニアへの依頼内容が明確になった」「業務改善のアイデアが具体的に湧いてくるようになった」といった声が多数寄せられています。

## 3. 到達レベル：本格SNSアプリが作れる技術力

### なぜ「SNS開発」がゴールなのか？

本ブートキャンプでは、最終ゴールとして「オリジナルのSNSを開発し、公開できるレベル」を設定しています。なぜなら、SNS開発という題材には、現代のWebアプリケーション開発に求められるフルスタックの技術要素が網羅的に含まれているからです。

- **ユーザー視点の想像力と要件定義能力**: どのような機能があればユーザーが喜ぶかを想像し、それを具体的な要件に落とし込む力が養われます。
- **実践的な設計・実装スキル**: 投稿、いいね、フォロー、コメントといった機能を実装する過程で、UI/UX設計、データベース設計、API連携など、実践的な開発スキルが体系的に身につきます。
- **フルスタック技術の網羅**: フロントエンド（Next.js, React）、バックエンド（API, サーバーアクション）、データベース（Supabase）、認証（Clerk）、デプロイ（Vercel）まで、一連の開発プロセスをすべて経験できます。

身近なテーマでありながら、サービス開発の勘所を掴む上で、SNSは最適な学習題材です。

### 技術スタックと設計力（完全伴走型の到達ライン）

| 領域               | 代表スキル・知識                                                     |
| ---------------- | ------------------------------------------------------------ |
| **フロントエンド**      | Next.js 14 + TypeScript / Tailwind / React Server Components |
| **バックエンド & API** | Next.js API Routes・Server Actions、外部 REST/GraphQL 統合  |
| **DB 設計**        | Supabase/PostgreSQL：RLS・全文検索・ER 図レベルのモデリング                   |
| **認証 / ストレージ**   | Clerk + Storage (画像・ファイル)                            |
| **CI/CD**        | Vercel 自動デプロイ    |
| **AI 駆動開発**      | Cursor / Claude Code でコード生成→リファクタ→テスト→デプロイを反復                 |

### 到達指標例
- 認証、投稿、いいね、フォローなどを含むSNSのコア機能を実装し、公開URLへデプロイ
- ER図を作成し、10以上のテーブルを持つデータベースをSupabase/PostgreSQLで設計・構築
- 外部API（例：GCPのAPIなど）を連携させた機能を1つ以上追加し、サービスの可能性を拡張

- Architectモードでの計画とCodeモードでの実装の分離
- コンテキストメンションの活用で必要情報を効率的に提供

## 10. 受講生に起きる「思考の変化」事例

本ブートキャンプがもたらすのは、単なる技術習得ではありません。AIを前提とした課題解決能力、いわば「ビジネスOSのアップデート」です。開発未経験だった方々が、わずか数週間で以下のような思考の変化を遂げています。

- **事例1：API連携によるサービス拡張の発想**
  - **受講前**：「APIって何？難しそう…」
  - **受講後**：「なるほど、GCPからこのAPIを持ってきて、あのサービスと繋げば、自分たちで新しい価値を創れるじゃないか！」と、既存の技術を組み合わせた新規事業のアイデアを自発的に議論し始める。

- **事例2：身近な業務課題の自動化**
  - **受講前**：大量の文字起こしデータを手作業で整理。
  - **受講後**：「この音声スクリプトをMarkdownで自動保管して、日付ごとに処理させれば、あの面倒な作業が一瞬で終わるのでは？」と、学んだスキルを即座に自身の業務改善に応用。

- **事例3：エンジニアとの円滑な連携**
  - **受講前**：エンジニアへの依頼が曖昧で、手戻りが多かった。
  - **受講後**：「こんな感じで、とER図とスキーマを渡せば、やりたいことが具体的に伝わりますよね。これならYes/Noで的確な判断をもらえる！」と、開発の共通言語を理解し、よりスムーズな協業を実現。

- **事例4：開発環境への完全な移行**
  - **受講前**：ChatGPTにファイルを一つずつドラッグ＆ドロップ。
  - **受講後**：「気づいたら普通にCursor（IDE）で作業してる。もうチャット画面には戻れないです。音声入力マイクも買って、AIに渡す情報量が圧倒的に増えました。」と、プロの開発者と同じ土俵でAIを使いこなす。

このように、Vibe Codingの本質は、AIを使いこなし、ビジネスを進化させる思考回路そのものなのです。
