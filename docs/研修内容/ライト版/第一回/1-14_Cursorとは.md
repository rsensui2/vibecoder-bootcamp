# Cursorとは

## 概要

CursorはAI機能が深く統合された次世代のコードエディタです。2025年6月4日にバージョン1.0がリリースされ、VS Codeをベースとしながら、**Agentモード（旧Composer）を中心とした自然言語プログラミング**を実現します。Vibe Codingの中核ツールとして、コードを書かずにアプリケーションを構築できる革新的な開発環境です。

## 学習目標

- CursorのAgentモード中心の操作方法を理解する
- 自然言語でアプリケーションを構築する手法を習得する
- Vibe Coding実践のための効果的なプロンプト作成を身につける
- 非エンジニアでも使える実践的な開発パターンを理解する

## 内容

### 1. Cursorの基本概念と最新アップデート

**Cursor 1.0の革新的変更点：**

- **統合されたインターフェース**：Chat、Composer、Agentが一つに統合
- **Agentモードがデフォルト**：自然言語での指示が基本操作に
- **Background Agent**：バックグラウンドで自動的にタスクを実行
- **YOLOモード**：AIが自律的にファイル作成から修正まで実行

**Vibe Coder向けの特徴：**

- プログラミング知識不要の**自然言語インターフェース**
- **スクリーンショットからコード生成**（Make Real機能）
- **エラーの自動修正**と継続的な改善
- **8歳の子供でも45分でアプリ作成可能**な直感性

**従来の開発との決定的な違い：**

- **従来**：コードの文法を覚えて一行ずつ記述
- **Cursor**：「こんなアプリを作って」と日本語で伝えるだけ
- **成果**：プログラミング未経験者が商用アプリを構築可能に

### 2. Agentモード（Composer）完全ガイド

**基本的な使い方：**

1. **Cmd+I**（Mac）または**Ctrl+I**（Windows）でAgentモードを開く
2. 作りたいものを**日本語で詳しく説明**
3. AIが**複数ファイルを自動生成**
4. **自動的にエラーを検出・修正**

**実践例 - ToDoアプリの作成：**

```
プロンプト例：
「シンプルなToDoアプリを作ってください。
- タスクの追加と削除ができる
- 完了したタスクにチェックマークをつけられる
- データはブラウザに保存される
- 見た目はモダンでスマホでも使いやすい」
```

**生成される成果物：**

- `index.html` - アプリの構造
- `app.js` - 機能の実装
- `styles.css` - デザイン
- `README.md` - 使い方の説明

### 3. Background Agentによる並列開発

**革新的な新機能：**

- **リモートAWS環境で実行**：ローカルPCの負荷なし
- **並列処理**：複数のタスクを同時に実行
- **自動テスト実行**：作成したコードの動作確認
- **継続的な改善**：バグを見つけて自動修正

**活用シナリオ：**

```
「先ほど作ったToDoアプリに以下の機能を追加して：
- カテゴリー分け機能
- 期限設定
- 優先度の色分け
バックグラウンドで実行してください」
```

### 4. Vibe Coding実践テクニック

**効果的なプロンプトの書き方：**

**具体的で詳細な指示：**

```
❌ 悪い例：「かっこいいサイト作って」

✅ 良い例：
「個人ポートフォリオサイトを作成してください。
- 自己紹介セクション（写真付き）
- 作品ギャラリー（画像をクリックで拡大）
- お問い合わせフォーム（メール送信機能）
- ダークモード切り替え
- スマホ対応のレスポンシブデザイン」
```

**段階的アプローチ：**

1. **基本機能から始める**：最小限の機能で動くものを作る
2. **機能を追加**：一つずつ機能を追加していく
3. **デザインを改善**：見た目を洗練させる
4. **最適化**：パフォーマンスや使いやすさを向上

### 5. スクリーンショットを使った開発

**Make Real機能の活用：**

1. デザインツールやペイントで**UIのイメージを作成**
2. スクリーンショットを**Cursorにドラッグ&ドロップ**
3. 「**このデザインを実装して**」と指示
4. AIが**デザインを解析してコード生成**

**成功事例：**

- 手書きスケッチから機能するWebアプリを15分で作成
- Figmaデザインから完全なReactコンポーネントを生成
- 既存サイトのスクリーンショットから類似サイトを構築

### 6. 実践的なプロジェクト例

**15分で作れるアプリケーション：**

**1. 個人ブログ：**

```
「マークダウンで記事を書けるシンプルなブログを作って。
記事一覧と個別記事ページ、タグ機能付き」
```

**2. 経費精算アプリ：**

```
「経費を記録できるアプリを作成。
カテゴリー別集計とグラフ表示機能付き」
```

**3. 習慣トラッカー：**

```
「毎日の習慣をチェックできるアプリ。
カレンダー表示と達成率のグラフ付き」
```

### 7. YOLOモードでの自動開発

**完全自動化の実現：**

- **ファイル作成から実行まで自動化**
- **エラーを検出して自動修正**
- **最適なフォルダ構造を自動決定**
- **必要なライブラリを自動インストール**

**使用方法：**

```
プロンプト：
「YOLOモードで実行：
オンラインホワイトボードアプリを作成。
複数人で同時に絵を描ける機能付き」
```

### 8. エラー対処とトラブルシューティング

**AIが自動的に解決すること：**

- 文法エラーの修正
- 未定義変数の検出と修正
- インポート文の自動追加
- 非推奨メソッドの更新

**人間が確認すべきこと：**

- ビジネスロジックの正確性
- セキュリティ要件の確認
- パフォーマンス要件の達成
- ユーザビリティの確認

## 重要ポイント

- **コードを見ない・触らない**：Agentモードで自然言語のみで開発
- **具体的な指示**：作りたいものを詳細に日本語で説明
- **段階的な改善**：完璧を求めず、少しずつ機能追加
- **AIを信頼する**：YOLOモードで自動実行を活用
