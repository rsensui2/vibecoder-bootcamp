## 1. GitHub（無料・最優先）

### なぜ最優先？
- Cursor Proのログインに**GitHubアカウントが必要**
- v0.dev、Vercel等の多くのサービスがGitHub連携を使用
- コード管理の中心となるプラットフォーム

### 作成手順

#### ステップ1: アカウント登録
1. [https://github.com](https://github.com) にアクセス
2. 右上の **「Sign up」** をクリック
3. 以下の情報を入力：
   - メールアドレス（推奨：個人メール）
   - パスワード（強力なパスワードを設定）
   - ユーザー名（半角英数字、ハイフン使用可）
4. 「Verify your account」でパズルを解く
5. 「Create account」をクリック

#### ステップ2: メール認証
1. 登録したメールアドレスに届いた確認メールを開く
2. 8桁の確認コードをコピー
3. GitHub画面に戻り、確認コードを入力

#### ステップ3: 初期設定
1. アンケートが表示される（スキップ可能）
2. Freeプランを選択（デフォルト）
3. ダッシュボードに移動

#### ステップ4: 2段階認証の設定（重要）
1. 右上のアイコンをクリック → **Settings**
2. 左メニューから **Password and authentication**
3. **「Enable two-factor authentication」** をクリック
4. 認証方法を選択：
   - **Authenticatorアプリ**（推奨）：Google Authenticator、Microsoft Authenticator等
   - **SMS**：電話番号で認証コード受信
5. 指示に従って設定を完了

### ✅ 確認方法
- GitHubにログインできる
- プロフィールページ（`https://github.com/あなたのユーザー名`）が表示される
- 2段階認証が有効になっている

## 2. Cursor Pro（有料・最優先）

### なぜ最優先？
- AI駆動開発の**中心ツール**
- Composerエージェントによる自動環境構築
- 講義で使用する主要なコードエディタ

### 作成手順

#### ステップ1: Cursorのダウンロードとインストール

**macOSの場合**
1. [https://cursor.com](https://cursor.com) にアクセス
2. **「Download for Mac」** をクリック
   - Apple Silicon（M1/M2/M3）とIntelを自動判定
3. ダウンロードした `.dmg` ファイルをダブルクリック
4. Cursorアイコンを **Applications** フォルダにドラッグ&ドロップ
5. Applicationsフォルダから「Cursor」を起動
   - 「開発元を検証できません」と表示された場合：
     - 「システム設定」→「プライバシーとセキュリティ」→「このまま開く」

**Windowsの場合**
1. [https://cursor.com](https://cursor.com) にアクセス
2. **「Download for Windows」** をクリック
3. ダウンロードした `.exe` ファイルを実行
4. インストールウィザードに従って進める
   - 「Add to PATH」にチェックを入れる（推奨）
5. インストール完了後、Cursorを起動

#### ステップ2: GitHubでログイン
1. Cursorを起動すると、ログイン画面が表示される
2. **「Sign in with GitHub」** をクリック
3. GitHubの認証画面が開く
4. **「Authorize Cursor」** をクリック
5. 2段階認証コードを入力（設定している場合）
6. Cursorにリダイレクトされ、ログイン完了

#### ステップ3: Proプランにアップグレード
1. Cursor左下の **⚙️（設定アイコン）** をクリック
2. **「Billing」** を選択
3. **「Upgrade to Pro」** をクリック
4. 支払い情報を入力：
   - クレジットカード情報
   - 請求先住所
5. **「Subscribe」** をクリック

### 💡 Cursor Proの主な特典
- 無制限のTab補完（コード予測）
- 拡張されたAgent使用制限
- Background Agents（バックグラウンドでコード分析）
- Bugbot（バグ自動検出）
- 最大コンテキストウィンドウ
- レート制限あり

> 📌 **最新情報**: [公式機能説明](https://cursor.com/pricing)で詳細をご確認ください。

### ✅ 確認方法
1. Cursor左下の設定アイコン → **Billing**
2. 「Pro」と表示されている
3. 使用量メーターが表示される

## ❓ よくある質問（FAQ）

### Q1: クレジットカードを持っていません
**A**: Cursor ProとChatGPT Plusは有料サービスのため、クレジットカードが必要です。以下の代替案があります：
- デビットカード（Visaデビット等）が使用可能
- PayPalアカウントで支払い（一部サービス）
- 家族や知人に相談して支払いを依頼

### Q2: 学生割引はありますか？
**A**:
- **Cursor**: 学生割引プランあり。GitHub Student Developer Packで申請可能。
- **ChatGPT Plus**: 学生割引なし。
- **GitHub**: GitHub Student Developer Pack（無料特典多数）

> 📌 最新の学生割引情報は各サービスの公式サイトでご確認ください。

### Q3: 無料トライアルはありますか？
**A**:
- **Cursor**: 2週間の無料トライアルあり（要クレジットカード登録）
- **ChatGPT Plus**: 無料トライアルなし（無料版は常に利用可能）

### Q4: GitHubアカウントの2段階認証は必須ですか？
**A**: 必須ではありませんが、**強く推奨**します。GitHubアカウントは多くのサービスと連携するため、セキュリティが非常に重要です。

### Q5: 会社のメールアドレスを使うべきですか？
**A**: 個人のメールアドレスを推奨します。将来転職した場合もアカウントを継続使用できます。

### Q6: v0.devの$5クレジットはいつリセットされますか？
**A**: 毎月1日に自動的にリセットされます。使い切っても翌月まで待てば再度使用可能です。

### Q7: Supabaseのデータベースパスワードを忘れました
**A**: Supabaseのダッシュボードから再設定できます：
1. Settings → Database
2. 「Reset database password」をクリック

### Q8: Vercel Hobbyプランで商用サイトは公開できますか？
**A**: できません。Hobbyプランは非商用のみです。商用サイトはProプラン（有料）が必要です。

> 📌 最新のプラン情報は[Vercel公式サイト](https://vercel.com/pricing)でご確認ください。

## 🆘 トラブルシューティング

### GitHub認証でエラーが出る
**症状**: 「Authentication failed」と表示される
**解決策**:
1. ブラウザのキャッシュをクリア
2. シークレットモード/プライベートブラウジングで試す
3. 別のブラウザ（Chrome、Firefox、Safari）で試す
4. 2段階認証コードを正しく入力しているか確認

### Cursorのインストールが失敗する（macOS）
**症状**: 「開発元を検証できません」
**解決策**:
1. 「システム設定」→「プライバシーとセキュリティ」
2. 下部の「このまま開く」をクリック
3. パスワードを入力して許可

### Cursorのインストールが失敗する（Windows）
**症状**: 「WindowsによってPCが保護されました」
**解決策**:
1. 「詳細情報」をクリック
2. 「実行」をクリック
3. インストールを続行

### クレジットカードが拒否される
**症状**: 「Payment failed」
**解決策**:
1. カード情報（番号、有効期限、CVV）を再確認
2. 請求先住所がカード登録住所と一致しているか確認
3. 別のクレジットカードを試す
4. デビットカードやPayPalを試す
5. カード会社に海外決済が拒否されていないか確認

### メール認証コードが届かない
**症状**: 確認メールが受信できない
**解決策**:
1. 迷惑メールフォルダを確認
2. メールアドレスのスペルミスがないか確認
3. 5〜10分待ってから再送信を試す
4. メールプロバイダーの受信設定を確認
