# presentation_design_spec_sculpture_pop_jp.yaml
# スタイル: 彫刻ポップアート / ヴェイパーウェイヴ / シュルレアリスム・コラージュ
# コンセプト: 「古典（クラシック）と現代ポップの融合」

```yaml
全体デザイン設定:
  テーマ: "ヴェイパーウェイヴ ＆ ポップアート・シュルレアリスム"
  トーン: "大胆、遊び心、アーティスティック、高コントラスト、目を引く"
  ビジュアル・アイデンティティ:
    背景色: "ビビッドな単色 (エレクトリックパープル #9B59B6, サンシャインイエロー #F1C40F, ホットピンク #FF69B4)"
    画像要素:
      被写体: "古典的な大理石の彫刻（ギリシャ・ローマの胸像）"
      アクセサリー: "現代的なポップアイテム（サングラス、ヘッドフォン、風船ガム、ソーダカップ）を明るい色で"
      スタイル: "リアルな3Dレンダリング、または高品質なフォトコラージュ"
    タイポグラフィ:
      見出し: "太字のサンセリフ体（Helvetica Bold, Futura）または レトロなセリフ体（Cooper Black）"
      本文: "クリーンでミニマルなサンセリフ体（視覚的な賑やかさとバランスを取るため）"
      文字色: "白 または 黒（ビビッドな背景に対する高コントラスト）"

スライド構成:
  - スライド番号: 1
    セクション: "表紙 / タイトル"
    レイアウトタイプ: "センター・ヒーロー"
    デザイン詳細:
      - "背景: エレクトリックパープル (#9B59B6)。"
      - "ヒーロー画像: 明るい黄色のサングラスをかけ、オレンジのカップで飲み物を飲む大理石の胸像。"
      - "タイトル: 画像に少し重なるように、巨大な白文字の太字タイトルを配置。"
      - "サブタイトル: その下に小さくシンプルに配置。"

  - スライド番号: 2
    セクション: "コンセプト / アイデア"
    レイアウトタイプ: "スプリットスクリーン・コントラスト"
    デザイン詳細:
      - "背景: サンシャインイエロー (#F1C40F)。"
      - "画像: カラフルな丸サングラスをかけて叫んでいる（または歌っている）胸像のアップ。"
      - "テキストエリア: 黄色い背景に黒の太字テキスト。白い吹き出しやボックスに入れて配置。"
      - "レイアウト: 片側に画像、反対側にテキストを置き、動的なバランスを作る。"

  - スライド番号: 3
    セクション: "特徴 / プロダクト"
    レイアウトタイプ: "アクセサリー・フォーカス"
    デザイン詳細:
      - "背景: ホットピンク (#FF69B4)。"
      - "ヒーロー画像: ピンクの風船ガムを膨らませている胸像。"
      - "コンテンツ: 白いシンプルなアイコンを使った箇条書きテキスト。"
      - "視覚効果: 風船ガムの「円形」と、箇条書きの「丸い行頭文字」を視覚的にリンクさせる。"

  - スライド番号: 4
    セクション: "テクノロジー / リスニング"
    レイアウトタイプ: "モダン・インテグレーション"
    デザイン詳細:
      - "背景: オレンジを際立たせるための、少し深みのあるトーン。"
      - "ヒーロー画像: 明るいオレンジのヘッドフォンを装着し、スマートフォンを持つ胸像。"
      - "テキスト: 「Listen」「Connect」などのキーワードを、ヘッドフォンと同じオレンジ色の巨大文字で配置。"

  - スライド番号: 5
    セクション: "データ / インパクト"
    レイアウトタイプ: "ポップ・チャート"
    デザイン詳細:
      - "背景: ビビッドなブルー または グリーン。"
      - "チャート: アクセサリーに使われているポップカラー（ピンク、黄、オレンジ）を使ったシンプルな棒グラフや円グラフ。"
      - "装飾: 「彫刻の手」だけを切り抜いて配置し、データを指差すような演出を加える。"

デザインルール:
  - "被写体の分離": "彫刻とアクセサリーは、単色の背景からきれいに切り抜いて配置する（Isolate）。"
  - "色のマッチング": "テキストのアクセントカラーをアクセサリーの色に合わせる（例：サングラスが黄色ならテキストも黄色）。"
  - "ユーモア": "現代的なオブジェクトとコミカルに相互作用している彫刻のポーズを選ぶ。"
  - "テキスト最小化": "ビジュアルのインパクトを最優先するため、文字は短くパンチの効いたものにする。"
```
