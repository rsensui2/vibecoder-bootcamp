## AIエージェントへの指示

このディレクトリの運用ルール：

### 追記時のルール
1. **カテゴリー判断**: 追加する情報がどのカテゴリーに属するか判断
2. **既存確認**: 同じ内容が既に記載されていないか確認
3. **適切なセクションに追記**: 各ファイルの構造に従って適切なセクションに追記
4. **具体例を含める**: コマンド例、コードスニペット、スクリーンショット説明などを含める
5. **初心者目線**: 専門用語は必ず説明を添える

### フォーマット規則
- 見出しレベルは適切に使用（##, ###, ####）
- コードブロックは言語指定（```bash, ```typescript など）
- 重要な情報は **太字** や `コード記法` で強調
- リストは `-` を使用
- 番号付きリストは `1.` を使用

### 追記の優先度
- **高**: 複数の受講生から質問があった内容
- **中**: 重要だがあまり質問されない内容
- **低**: 補足的な情報

### 情報の鮮度管理
- 各ファイルの最上部に「最終更新日」を記載
- バージョン情報が関係する場合は明記（例: Next.js 14, Cursor v0.42など）
